## Product Identity
Enlivonex AI Hub is an AI Workspace that brings multiple AI tools into one customizable, modern and user-friendly platform.


## User Flow

1. User opens Enlivonex AI Hub.

2. User lands on the Home Page.

3. User clicks "Get Started".

4. Dashboard opens.

5. User selects an AI tool.

6. User uses the selected tool.

7. Results are displayed.

8. History is saved.

9. User exits the application.
