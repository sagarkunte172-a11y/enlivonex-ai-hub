# 🚀 Enlivonex AI Hub

> One Workspace. Every AI. Every Workflow.

## 🌟 Vision

Enlivonex AI Hub is an AI productivity platform designed to bring multiple AI models and intelligent workflows into one unified workspace.

Instead of switching between different AI tools for text, images, videos, coding, research, and productivity, users can access everything from a single platform.

## 🎯 Goals

- AI Chat
- AI Image Generation
- AI Video Generation
- AI Script Generator
- AI Research Assistant
- AI Coding Assistant
- AI PDF Tools
- AI Workflow Automation
